# Text

Add a text message to your blog.

## Usage

```xml
<b:widget id='Text1' locked='false' title='Text' type='Text' visible='true'>
  <b:widget-settings>
    <b:widget-setting name='content'><![CDATA[ content ]]></b:widget-setting>
  </b:widget-settings>
</b:widget>
```

### Example

```xml
<b:widget id='Text1' locked='false' title='Text' type='Text' visible='true'>
  <b:widget-settings>
    <b:widget-setting name='content'>
      <![CDATA[
        Hello, world!
      ]]>
    </b:widget-setting>
  </b:widget-settings>
</b:widget>
```
